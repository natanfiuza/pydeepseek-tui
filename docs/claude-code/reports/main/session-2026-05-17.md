## 2. Bugfixes

| Problema | Solução |
|----------|---------|
| `prompt_preview` sempre igual (22x mesmo texto) | Nomes das tools em rounds tool-call |
| `response_preview` vazio | Fallback para nomes das tools |
| Confirmação não-interativa (`_tui_confirm` sempre False) | `ConfirmScreen` modal com `push_screen` + `asyncio.Future` |
| `_trim_history` quebrava pares assistant/tool (erro 400) | Remove tool messages órfãs após truncagem |
| Respostas textuais não guardadas no histórico | Adiciona `assistant` msg com `reasoning_content` |
| `debug.log` com um chunk por linha de OUTPUT | Acumula chunks, grava resposta completa numa linha |
| `manifest.json` sobrescrito a cada sessão | Array com append; `current_session_id` global |
