# LIsta de comandos `make`

| Comando           | O que faz                          |
| ----------------- | ---------------------------------- |
| make install-dev  | Instala tudo via pipenv            |
| make run          | Executa a TUI                      |
| make dev          | TUI com hot-reload Textual         |
| make config       | Assistente de configuração         |
| make check        | Lint + format + typecheck          |
| make test-cov     | Testes com relatório HTML          |
| make build        | Gera wheel + sdist                 |
| make publish-test | Publica no TestPyPI                |
| make publish      | Publica no PyPI (pede confirmação) |
| make bump-patch   | 0.1.0 → 0.1.1 automático           |
| make info         | Status do ambiente e config        |
